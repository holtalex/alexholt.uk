# alexholt.uk
My personal portfolio site - built to be bold, minimal, and just a little bit different.
 
Take a look: [www.alexholt.uk](https://alexholt.uk)

## What
A single-page portfolio site showing you a bit about me, what I like, and what I've done. The design is incredibly bold, and certainly not subtle, but (I hope) still retains a sense of elegance to it. The font is clean, colours striking, and some subtle animations add that bit of *je ne sais quoi*.

## Features
- Big, bold typeface - especially in the hero section
- Fun hover animations for text on desktop
- Unusual vertical text in the hero section on mobile
- Full support for both light and dark modes
- Semantic HTML for improved accessibility
- Pure HTML & CSS
